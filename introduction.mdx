## What is Wildlife Watcher?

The Wildlife Watcher mobile app is a React Native application that enables users to communicate with Wildlife Watcher cameras via Bluetooth Low Energy (BLE). These cameras record wildlife activity and use AI to identify animals in the field.

The Wildlife Watcher mobile app provides comprehensive tools for wildlife monitoring:

## Technology Stack

Built with modern mobile development technologies:

- **React Native 0.74.6** - Cross-platform mobile framework
- **Redux Toolkit** - State management with RTK Query for API calls
- **React Navigation** - Native navigation with bottom tabs
- **React Native Paper** - Material Design component library
- **BLE Manager** - Bluetooth Low Energy communication
- **Firebase** - Authentication and app distribution
