## Quick Start

Get the app running on your device in minutes.

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install Node.js dependencies
    ```bash
    git clone https://github.com/wildlifeai/wildlife-watcher-mobile-app.git
    cd wildlife-watcher-mobile-app
    npm install
    ```
  </Step>
  <Step title="Set Up Platform">
    For iOS, install CocoaPods dependencies:
    ```bash
    bundle install
    bundle exec pod install --project-directory=ios
    ```
    
    For Android, ensure you have Android Studio and SDK set up per the [React Native documentation](https://reactnative.dev/docs/set-up-your-environment).
  </Step>
  <Step title="Run the App">
    Start Metro bundler and launch the app:
    ```bash
    npm start
    npm run ios    # For iOS
    npm run android # For Android
    ```
  </Step>
</Steps>

## Project Overview

Watch the [Wildlife Watcher project overview on YouTube](https://www.youtube.com/watch?v=Ima3n2EYfeE) to learn more about the wildlife monitoring system.

<Note>
  This mobile app is part of the Wildlife Watcher ecosystem, which includes AI-powered wildlife cameras that detect and identify animals in the field. The app enables field researchers to configure cameras, download data, and manage deployments without requiring internet connectivity.
</Note>
