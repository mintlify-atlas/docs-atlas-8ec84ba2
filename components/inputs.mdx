## WWTextInput

A text input field with built-in validation, error handling, and icon support.

### Props

<ParamField path="value" type="string">
  Current value of the input field
</ParamField>

<ParamField path="onChange" type="(value: string) => void">
  Callback function called when text changes
</ParamField>

<ParamField path="placeholder" type="string">
  Placeholder text displayed when input is empty
</ParamField>

<ParamField path="label" type="string">
  Label text displayed above/inside the input
</ParamField>

<ParamField path="multiline" type="boolean" default="false">
  Allows multiple lines of text input
</ParamField>

<ParamField path="hasError" type="boolean" default="false">
  When true, displays error styling and icon
</ParamField>

<ParamField path="errorText" type="string">
  Error message displayed below the input when `hasError` is true
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the input and shows a lock icon
</ParamField>

<ParamField path="secureTextEntry" type="boolean" default="false">
  Masks input text (for passwords)
</ParamField>

<ParamField path="right" type="React.ReactNode">
  Custom right-side icon or component
</ParamField>

<ParamField path="style" type="StyleProp<ViewStyle>">
  Custom styles for the input container
</ParamField>

### Type Definition

```tsx
export type WWTextInputProps = CommonProps & Omit<TextInputProps, 'onChange'>

type CommonProps = {
  multiline?: boolean
  hasError?: boolean
  onChange?: (value: string) => void
  errorText?: string
}
```

## WWSelect

A dropdown selection component for choosing from a list of options.

### Props

<ParamField path="label" type="string" required>
  Label text for the dropdown
</ParamField>

<ParamField path="options" type="Option[]" required>
  Array of options with `label` and `value` properties
</ParamField>

<ParamField path="value" type="string">
  Currently selected value
</ParamField>

<ParamField path="onChange" type="(value: string) => void">
  Callback function called when selection changes
</ParamField>

<ParamField path="onSelectEffect" type="(value: string) => void">
  Additional callback for side effects when selection changes
</ParamField>

<ParamField path="hasError" type="boolean" default="false">
  When true, displays error styling
</ParamField>

<ParamField path="errorText" type="string">
  Error message displayed below the dropdown when `hasError` is true
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the dropdown
</ParamField>

<ParamField path="style" type="StyleProp<ViewStyle>">
  Custom styles for the dropdown container
</ParamField>

### Type Definition

```tsx
export type WWSelectProps = CommonProps & Omit<DropdownProps, 'onSelect' | 'options'>

export type Option = {
  label: string
  value: string
}

type CommonProps = {
  hasError?: boolean
  onChange?: (value: string) => void
  onSelectEffect?: (value: string) => void
  errorText?: string
  label: string
  options: Option[]
  style?: StyleProp<ViewStyle>
}
```

## WWCheckbox

A checkbox input component with label support.

### Props

<ParamField path="value" type="boolean" default="false">
  Current checked state of the checkbox
</ParamField>

<ParamField path="onChange" type="(value: boolean) => void">
  Callback function called when checkbox state changes
</ParamField>

<ParamField path="label" type="string">
  Label text displayed next to the checkbox
</ParamField>

<ParamField path="hasError" type="boolean" default="false">
  When true, displays checkbox in error color (red)
</ParamField>

<ParamField path="errorText" type="string">
  Error message (not displayed in current implementation)
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the checkbox
</ParamField>

<ParamField path="style" type="StyleProp<ViewStyle>">
  Custom styles for the checkbox container
</ParamField>

### Type Definition

```tsx
type Props = {
  style?: StyleProp<ViewStyle>
  hasError?: boolean
  errorText?: string
  value?: boolean
  onChange?: (value: boolean) => void
  label?: string
} & Omit<CheckboxProps, 'status' | 'onPress'>
```

### Automatic Icons

WWTextInput automatically displays:
- Lock icon when `disabled` is true
- Alert circle icon (red) when `hasError` is true
- Custom icon when `right` prop is provided

### Form Integration

All input components work seamlessly with form libraries like React Hook Form:

```tsx
import { useForm, Controller } from 'react-hook-form'
import { WWTextInput, WWSelect, WWCheckbox } from '@/components/ui'

export const FormExample = () => {
  const { control, handleSubmit } = useForm()
  
  return (
    <>
      <Controller
        control={control}
        name="species"
        render={({ field }) => (
          <WWTextInput
            label="Species"
            value={field.value}
            onChange={field.onChange}
          />
        )}
      />
    </>
  )
}
```
