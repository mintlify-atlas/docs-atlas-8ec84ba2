## WWButton

The primary button component with built-in error state handling. Based on React Native Paper's Button component.

### Props

<ParamField path="hasError" type="boolean" default="false">
  When true, applies error styling (red border) to the button
</ParamField>

<ParamField path="errorText" type="string">
  Error message displayed below the button when `hasError` is true
</ParamField>

<ParamField path="mode" type="'text' | 'outlined' | 'contained' | 'elevated' | 'contained-tonal'">
  Button visual style from React Native Paper
</ParamField>

<ParamField path="onPress" type="() => void">
  Callback function triggered when the button is pressed
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the button and prevents user interaction
</ParamField>

<ParamField path="icon" type="string | React.ReactNode">
  Icon to display in the button (Material Community Icons name or custom component)
</ParamField>

<ParamField path="loading" type="boolean" default="false">
  Shows a loading spinner inside the button
</ParamField>

<ParamField path="children" type="React.ReactNode">
  Button label text or content
</ParamField>

<ParamField path="style" type="StyleProp<ViewStyle>">
  Custom styles for the button
</ParamField>

### Type Definition

```tsx
export type WWButtonProps = CommonProps & ButtonProps

type CommonProps = {
  hasError?: boolean
  errorText?: string
}
```

## WWIconButton

A compact icon-only button for toolbar actions and compact UI elements.

### Props

<ParamField path="icon" type="string" required>
  Material Community Icons icon name
</ParamField>

<ParamField path="onPress" type="() => void">
  Callback function triggered when the button is pressed
</ParamField>

<ParamField path="disabled" type="boolean" default="false">
  Disables the button and prevents user interaction
</ParamField>

<ParamField path="size" type="number" default="24">
  Size of the icon in pixels
</ParamField>

<ParamField path="iconColor" type="string">
  Custom color for the icon
</ParamField>

<ParamField path="style" type="StyleProp<ViewStyle>">
  Custom styles for the button container
</ParamField>

### Type Definition

```tsx
export type WWIconButtonProps = CommonProps & IconButtonProps

type CommonProps = {
  hasError?: boolean
  errorText?: string
}
```

### Error Styling

When `hasError` is true, buttons receive:
- Red border (1px width)
- Error text displayed below in red color (12px font size)
- 4px top margin for error text

### Accessibility

Both button components:
- Support ref forwarding
- Inherit accessibility props from React Native Paper
- Provide visual feedback for disabled states
