## WWText

A styled text component with theme integration and convenience props.

### Props

<ParamField path="children" type="React.ReactNode" required>
  Text content to display
</ParamField>

<ParamField path="variant" type="string" default="bodyMedium">
  Typography variant from React Native Paper theme:
  - `displayLarge`, `displayMedium`, `displaySmall`
  - `headlineLarge`, `headlineMedium`, `headlineSmall`
  - `titleLarge`, `titleMedium`, `titleSmall`
  - `bodyLarge`, `bodyMedium`, `bodySmall`
  - `labelLarge`, `labelMedium`, `labelSmall`
</ParamField>

<ParamField path="align" type="'left' | 'center' | 'right' | 'justify'">
  Text alignment
</ParamField>

<ParamField path="gutter" type="boolean" default="false">
  Adds bottom margin using theme spacing
</ParamField>

<ParamField path="style" type="StyleProp<TextStyle>">
  Custom text styles
</ParamField>

### Type Definition

```tsx
type Props = TextProps<never> & {
  align?: TextStyle['textAlign']
  gutter?: boolean
}
```

## WWIcon

Icon display component using Material Community Icons.

### Props

<ParamField path="source" type="string | React.ReactNode" required>

</ParamField>

<ParamField path="size" type="number" required>
  Size of the icon in pixels
</ParamField>

<ParamField path="color" type="string">
  Icon color (defaults to theme color)
</ParamField>

<ParamField path="containerStyle" type="StyleProp<ViewStyle>">
  Custom styles for the icon container
</ParamField>

<ParamField path="allowFontScaling" type="boolean">
  Whether the icon should scale with font size settings
</ParamField>

<ParamField path="testID" type="string">
  Test identifier for automated testing
</ParamField>

## WWProgressBar

A progress bar with optional label display.

### Props

<ParamField path="progress" type="number" default="0">
  Progress value between 0 and 1 (e.g., 0.5 for 50%)
</ParamField>

<ParamField path="showLabel" type="boolean" default="false">
  Whether to display the progress label
</ParamField>

<ParamField path="label" type="string">
  Custom label text (defaults to "Progress: X%")
</ParamField>

<ParamField path="color" type="string">
  Progress bar color (defaults to theme primary color)
</ParamField>

<ParamField path="style" type="StyleProp<ViewStyle>">
  Custom styles for the progress bar
</ParamField>

### Type Definition

```tsx
type Props = ProgressBarProps & {
  showLabel?: boolean
  label?: string
}
```

### Usage Examples

<Tabs>

import { WWProgressBar } from '@/components/ui'

    }, 500)

  return (

      showLabel
    />
  )

</ParamField>

<ParamField path="onPress" type="() => void">

## Implementation Details

### Source Code Locations

- **WWText**: `/workspace/source/src/components/ui/WWText.tsx:11-33`
- **WWIcon**: `/workspace/source/src/components/ui/WWIcon.tsx:26-32`
- **WWLoader**: `/workspace/source/src/components/ui/WWLoader.tsx:5-13`
- **WWProgressBar**: `/workspace/source/src/components/ui/WWProgressBar.tsx:12-33`
- **WWAvatar**: `/workspace/source/src/components/ui/WWAvatar.tsx:15-29`

### Theme Integration

#### WWText
- Uses `useExtendedTheme` to access `spacing` for gutter margins
- Inherits all typography variants from React Native Paper theme

#### WWLoader
- Uses theme `colors.primary` for the loading indicator color
- Centered with `flex: 1`, `justifyContent: 'center'`, `alignItems: 'center'`

#### WWProgressBar
- Uses theme `colors.primary` as default progress color
- Fixed height of 6px with 3px border radius
- Percentage calculated automatically: `Math.round(progress * 100)`

#### WWAvatar
- Displays static avatar image from `../../assets/avatar.png`
- Applies 6px end margin for spacing in layouts
- Conditionally wraps in `TouchableOpacity` when `onPress` is provided

### Accessibility

- **WWText**: Inherits all accessibility props from React Native Paper's Text
- **WWIcon**: Supports `testID` for automated testing
- **WWLoader**: ActivityIndicator has built-in accessibility announcements
- **WWProgressBar**: Progress percentage announced automatically
- **WWAvatar**: Touchable when `onPress` is provided for better UX
